# background_stt

An flutter plugin that runs Speech-to-Text in background. (For Android Only)

## Install
In your pubspec.yaml

```yaml
dependencies:
  background_stt: [LATEST_VERSION]
```

```dart
import 'package:background_stt/background_stt.dart';
```
